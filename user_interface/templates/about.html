Instructions

Search - To use the search page, simply enter a value that you would like to search for. This search will return all documents from the database that contain the search value.

The fields within a document that can be searched for are Sample, Sample material, and Experiment. Multiple entries can be searched for by comma-separating the entries.

Alternatively, all documents in the database can be viewed by entering an empty search.


Insert - To use the insert page, simply drag and drop a data file onto the drop area, or click on the drop area to browse your files.

Expected format of data files:
The data files will be automatically converted to JSON files that can be inserted into the database if the following requirements are met:

  1. The header information begins the line after the line containing "CONFLUENCE"
  2. In the table containing the measurement data, the fields are separated by '^'
  3. The measurement values begin the line immediately after the line with the fields (e.g. "Alpha Counts", "Energy Range (MeV)", etc.)
  4. The measurement values within table rows are separated by '|'
  5. The corrected data comes after, but not necessarily immediately after, the line containing "Corrected"


Update - To use the update page, first find the id of the document to be updated. This should be displayed along with the other contents of the document when found in search.

Once this document has been found, you will be able to edit or remove each individual field in header information and measurement values. Once the desired changes have been made, click the update doc button at the bottom of the page. There is also a button to remove the entire document if one wants to remove the entire measurement or replace it with a later updated insert.


Login/logout - Login credentials are required to use any of the search, insert, or update features.
